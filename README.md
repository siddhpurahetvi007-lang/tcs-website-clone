# 🌐 TCS Website Clone (HTML & CSS)

A pixel-perfect clone of the official Tata Consultancy Services (TCS) website, built using only **HTML and CSS** to strengthen frontend fundamentals and UI precision.

## 🚀 Project Overview

This project replicates the design and layout of the TCS website with high accuracy.  
The main goal was to practice **pure HTML and CSS** by focusing on structure, styling, and layout without using JavaScript.

## 🛠️ Tech Stack

- HTML5 – Semantic structure  
- CSS3 – Styling and layout  

## ✨ Features

- Pixel-perfect UI matching the original website  
- Built entirely with HTML and CSS (no JavaScript)  
- Clean and structured layout  
- Responsive design (if implemented)  
- Flexbox and/or CSS Grid for alignment  
- Accurate spacing, typography, and design details
